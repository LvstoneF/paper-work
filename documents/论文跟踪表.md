# 综述论文跟踪表

> **用法**：每人把自己的 tab 复制到共享文档（飞书/腾讯文档），边搜边填。
> **核实前必跑**：《论文真实性校验提示词》的 6 项核对清单。
> **最后更新**：2025-06-22

---

## 使用说明

1. 每人维护自己的 sheet（见下方 A/B/C/D 四个 tab）
2. 搜到一篇填一行，**先占坑**（填简称+链接，核实状态标「核实中」）
3. 核实通过后，补全剩余字段，状态改「已核实」
4. 核实不通过，移到各自 tab 底部的「待核实区」
5. Day 6 合并时，所有「已核实」行汇总到「总池」

## 字段填写速查

| 字段 | 怎么填 | 示例 |
|------|--------|------|
| 论文简称 | 简短好记，5 字内更好 | VFArche |
| 完整标题 | 英文全标题，逐词核对 | VFArche: Version-Function Aware... |
| arXiv ID / DOI | 两者至少有一个 | 2506.18050 或 10.1145/3653718 |
| 年份 | 预印本看 Submitted on，出版版看出版年 | 2025 |
| 会议/期刊 | 全称缩写均可 | ICSE / USENIX / NeurIPS / arXiv |
| 方向 | ①②③④⑤⑥ | ② |
| 核心贡献 | 2–3 句自己的话，禁止复制摘要 | 提出版本+函数双维度定位，解决了... |
| 与综述相关性 | 1 句说清为什么有用 | 为方向②提供了新范式 |
| 核实状态 | 待搜 / 核实中 / 已核实 / 待核实 | 已核实 |
| 核实链接 | 你亲自打开过的完整 URL | https://arxiv.org/abs/2506.18050 |
| 备注 | 特殊说明、对比线索、存疑 | 与 Vercation 可对比写一段 |
